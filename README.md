# iOS Vision Framework Examples
This is a SwiftUI based iOS application that demonstrates Live Face Detection of Vision Frameworks usages.

## Features
- Live Face Detection (Saving mov and csv file)

## Library used:
- SwiftUI
- Vision
- AVFoundation
